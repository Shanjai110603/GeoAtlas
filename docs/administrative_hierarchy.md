## 4. Indian Regional Languages

### 🇮🇳 Tamil (ta)
- **மாநிலம் (State)** -> **மாவட்டம் (District)** -> **வட்டம் (Taluk)** -> **நகரம் (City)** -> **வார்டு (Ward)** -> **அஞ்சல் குறியீடு (PIN Code)** -> **தெரு (Street)**
