# GeoAtlas — 12-Level Worldwide Geographic Hierarchy Model

**Version:** 2.0

This document defines the complete **12-Level Geographic Knowledge Graph Hierarchy** for GeoAtlas, focusing on Asian countries and global standards.

---

## 1. Master 12-Level Hierarchy Tree

```text
Earth (Level 0)
 └─ Continents (Level 1 e.g. Asia, Europe, North America, South America, Africa, Oceania, Antarctica)
     └─ Subregions / Regions (Level 2 e.g. Southern Asia, Eastern Asia, South-Eastern Asia, Western Asia)
         └─ Countries (Level 3 - ADM0 e.g. India, China, Japan, Indonesia, Pakistan, Bangladesh, Philippines)
             └─ States / Prefectures / Provinces (Level 4 - ADM1)
                 └─ Districts / Counties / Regencies (Level 5 - ADM2)
                     └─ Taluks / Tehsils / Municipalities / Sub-districts (Level 6 - ADM3)
                         └─ Cities / Towns / Panchayats / Villages (Level 7 - ADM4)
                             └─ Neighborhoods / Wards / Barangays (Level 8)
                                 └─ Postal Codes (Level 9)
                                     └─ Streets & Roads (Level 10)
                                         └─ Buildings & Points of Interest (POIs) (Level 11)
                                             └─ Businesses, Hospitals, Schools (Level 12)
```
